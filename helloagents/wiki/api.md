# 用户入口与命令（非对外 API）

本项目为 Dalamud 插件，不提供任何对外 HTTP/API 服务。本文档仅记录对玩家可见的“使用入口”。

## 命令

### `/mp`
- **描述:** 打开/关闭主窗口。

## 分享与通报
- **手动播报:** 主窗口中可将“减伤巡查信息”发送到默语；如开启“同时发送到小队频道”，则会同步发送到小队频道。
- **自动通报（可选）:**
  - 冲突减伤（顶掉/覆盖）发生时自动播报（默认默语，可选同步小队）。
  - 有人死亡时自动播报一次减伤巡查信息（默认默语，可选同步小队），并包含致死详情（生前血/盾/致死伤害/过量）。

> 自动通报带有限速/去重逻辑以避免刷屏；小队频道发送为可选开关（默认关闭）。
